# DAX — Typography System (Light Mode)

# Purpose

This document defines the typography system for the DAX light mode experience.

Typography is not only communication.

Typography is:
- atmosphere
- composition
- rhythm
- scale
- emotion
- architecture
- visual storytelling

The typography system should feel:
- editorial
- cinematic
- luxurious
- futuristic
- spatial
- emotionally intentional

The website should feel designed around typography.

Not simply decorated with text.


---

# Core Typography Philosophy

Typography must behave like a visual structure.

Text should:
- shape the composition
- guide emotional pacing
- create hierarchy through scale
- establish atmosphere
- reinforce luxury perception

Avoid:
- generic startup typography
- small dense text blocks
- corporate hierarchy systems
- rigid spacing
- overly technical formatting

Typography should feel immersive.


---

# Emotional Typography Direction

The typography system should create feelings of:
- sophistication
- calm confidence
- futurism
- artistic elegance
- cinematic scale
- premium intelligence

The typography should feel emotionally composed.

Never mechanical.


---

# Editorial Influence

The typography direction should feel inspired by:
- luxury fashion editorials
- futuristic magazines
- cinematic title design
- architecture publications
- premium brand campaigns
- experimental visual design

Typography should feel curated by an art director.


---

# Scale Philosophy

Scale is one of the most important visual tools.

Use:
- oversized typography
- monumental headlines
- cinematic spacing
- dramatic scale contrast
- immersive visual rhythm

Large typography should establish emotional presence.

Avoid:
- conservative sizing
- overly safe hierarchy
- startup-style heading systems

The typography should feel bold and spatial.


---

# Hero Typography

Hero typography should feel:
- monumental
- iconic
- cinematic
- immersive
- emotionally powerful

The first headline should feel unforgettable.

It should dominate the visual atmosphere.

The hero text should behave like a visual sculpture inside the interface.


---

# Typography and Space

Typography must breathe.

Use:
- generous spacing
- cinematic line height
- elegant vertical rhythm
- intentional emptiness
- compositional balance

Whitespace around text is part of the typography system.


---

# Alignment Philosophy

Avoid:
- overly rigid centered layouts
- repetitive alignment systems
- predictable text placement

Use:
- editorial asymmetry
- floating alignment
- cinematic positioning
- visual rhythm
- intentional imbalance

Typography should feel composed.

Not templated.


---

# Font Personality

Fonts should feel:
- premium
- modern
- refined
- architectural
- intelligent
- timeless
- futuristic

Avoid:
- playful startup fonts
- overly geometric clichés
- trendy internet typography
- cartoonish type systems

Typography should feel globally sophisticated.


---

# Font Weight Philosophy

Weight should create elegance and rhythm.

Use:
- thin-to-bold contrast
- refined hierarchy
- confident emphasis
- selective boldness

Avoid:
- excessive heavy typography
- overly compressed dense text
- aggressive bold usage

Typography should feel balanced and breathable.


---

# Text Density

Avoid:
- large unreadable paragraphs
- dense information blocks
- overwhelming content structures

Use:
- short impactful statements
- cinematic pacing
- intentional content rhythm
- emotional spacing

Reading should feel immersive and effortless.


---

# Typography and Imagery

Typography should integrate with imagery.

Use:
- overlapping compositions
- environmental blending
- cinematic framing
- layered hierarchy
- interaction with light and space

Text and visuals should feel unified.


---

# Typography Motion

Typography movement should feel:
- elegant
- subtle
- cinematic
- spatial

Use:
- soft reveal timing
- atmospheric emergence
- smooth transitions
- floating movement

Avoid:
- aggressive kinetic typography
- flashy text effects
- social-media-style animation

Movement should reinforce sophistication.


---

# Contrast Philosophy

Contrast should feel controlled and cinematic.

Avoid:
- harsh black-on-white extremes
- aggressive visual separation
- cheap high-contrast aesthetics

Use:
- softened tonal relationships
- elegant luminosity
- atmospheric readability
- premium visual balance

Typography should feel integrated into the environment.


---

# Hierarchy Philosophy

Hierarchy should emerge through:
- scale
- spacing
- atmosphere
- composition
- rhythm
- light

Not excessive decoration.

Supporting text should feel:
- elegant
- lightweight
- atmospheric
- refined

- overly dark body text
- dense corporate copy

- soft
- premium
- atmospheric
- luminous

- warm neutrals
- softened dark tones
- elegant silver-gray relationships

Luxury typography is created through:
- restraint
- spacing
- confidence

- elegant
