# DAX — Color System (Light Mode)

# Purpose

This document defines the cinematic color system for the DAX light mode experience.

Color should not behave as decoration.

Color should behave as:
- atmosphere
- emotion
- light
- spatial depth
- visual rhythm
- environmental storytelling

The light mode color system must feel:
- premium
- futuristic
- atmospheric
- elegant
- cinematic
- emotionally controlled

The website should feel color graded like a film.


---

# Core Color Philosophy

The DAX light mode must never feel:
- flat white
- corporate
- sterile
- generic
- startup-like
- overly minimal
- overly colorful

Avoid:
- pure white backgrounds
- harsh black text
- loud gradients
- oversaturated palettes
- bright SaaS blues
- playful startup colors

The color system should feel sophisticated and emotionally restrained.


---

# Emotional Color Direction

The color palette should create feelings of:
- calm futurism
- elegance
- intelligence
- luxury
- visual softness
- atmospheric immersion

The environment should feel illuminated rather than colored.


---

# Cinematic Color Language

The visual atmosphere should feel inspired by:
- futuristic architecture
- luxury technology
- sci-fi cinema
- premium industrial design
- atmospheric photography
- soft editorial lighting

Colors should feel environmental.

Not graphic.
