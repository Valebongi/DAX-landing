# DAX — Animation & Scroll Engine (Light Mode)

# Purpose

This document defines the cinematic animation and scroll engine philosophy for the DAX light mode experience.

Animation should not feel:
- decorative
- gimmicky
- trendy
- startup-like
- attention-seeking

Animation should feel:
- cinematic
- immersive
- emotionally paced
- spatial
- atmospheric
- luxurious

Movement should create emotional continuity across the entire experience.


---

# Core Motion Philosophy

Motion is environmental.

The interface should feel:
- alive
- reactive
- spatially aware
- emotionally fluid

Movement should guide the user through atmosphere and visual storytelling.

Not through obvious animation tricks.


---

# Scroll Philosophy

Scrolling is one of the most important emotional systems of the website.

The user should feel like they are:
- moving through space
- entering different atmospheric zones
- progressing through cinematic scenes
- navigating an immersive environment

The website should never feel like stacked sections.


---

# Smooth Scroll System

Use:
- inertia-based smooth scrolling
- cinematic easing
- fluid progression
- responsive momentum
- atmospheric pacing

Preferred technologies:
- Lenis
- GSAP ScrollTrigger
- Framer Motion integration

Scrolling should feel elegant and premium.


---

# Scroll Pacing

Scrolling speed should feel:
- calm
- cinematic
- luxurious
- controlled

Avoid:
- overly fast movement
- abrupt transitions
- hyper-responsive startup scroll behavior

The user should feel guided through rhythm.


---

# Section Transition Philosophy

Sections should transition like cinematic scenes.

Use:
- environmental continuity
- atmospheric blending
- lighting continuity
- motion overlap
- gradual emergence

Avoid:
- hard transitions
- abrupt scene changes
- disconnected section reveals

The website should feel continuous.


---

# Reveal Animation Philosophy

Reveal animations should feel:
- atmospheric
- elegant

- subtle

- obvious fade-in systems

- trendy motion presets

Use:
- cinematic emergence
- soft opacity interpolation
- atmospheric movement

- subtle vertical drift

Reveals should feel naturally integrated into the environment.


---

# Motion Hierarchy

- selective emphasis
- layered motion priority
- focal animation
- restrained movement systems

Motion becomes powerful through contrast.
