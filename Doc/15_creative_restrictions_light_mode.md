# DAX — Creative Restrictions (Light Mode)

# Purpose

This document defines the creative restrictions for the DAX light mode experience.

These restrictions exist to protect:
- artistic integrity
- cinematic atmosphere
- luxury perception
- emotional immersion
- premium visual direction

The DAX experience must never collapse into:
- generic SaaS aesthetics
- startup visual clichés
- template-based design
- corporate interface patterns
- trend-driven UI systems

Every design decision must preserve the uniqueness of the experience.


---

# Core Restriction Philosophy

DAX should feel:
- globally elite
- cinematic
- futuristic
- immersive
- artistically directed
- emotionally intentional

Anything that weakens this perception must be avoided.


---

# Forbidden Visual Aesthetics

Never create:
- generic SaaS interfaces
- startup landing pages
- dashboard-style layouts
- corporate technology visuals
- obvious template structures
- common Webflow aesthetics
- repetitive modern UI clichés

The experience must feel custom-crafted and visionary.

# Luxury Restriction Philosophy

- desperation for attention
- excessive stimulation
- visual noise
- lack of restraint

Confidence creates sophistication.

Restraint creates luxury.

# Final Protection Principle

Every design decision should be evaluated with this question:

"Does this feel like a globally elite cinematic technology brand?"

If the answer is uncertain:
remove it.
